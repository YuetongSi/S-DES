# S-DES算法
本项目由司悦彤、李诗雨两位同学共同完成。实现了SDES(简化数据加密标准)对称密钥块加密算法，并提供一个图形用户界面。
## 功能
- 实现了SDES算法的加解密
- 提供图形用户界面进行交互
- 支持对ASCII字符串进行加解密
- 成功暴力破解给定的明密文对
- 成功分析明密文对是否存在多个可行密钥
